# ゲームデザイン文書

## 1. コンセプト

「エリーのアトリエ」にインスパイアされた、リソース・スケジュール管理に特化した錬金術シミュレーションゲーム。

### 省略要素
- 移動・探索（採取は抽象化）
- 戦闘
- キャラクターとの交流イベント

### 重視要素
- 時間（日数）管理
- 錬金術（調合）
- 依頼達成
- リソース管理
- **村発展**（新）

---

## 2. 世界観・ストーリー

### 背景
- 主人公は高名な錬金術士アルベルトの弟子
- 師匠から「旧友の村長が困っている。1年間行って手伝ってこい」と命じられる
- 舞台は辺境の寂れた村「ハイデル村」
- かつては交易で栄えたが、街道の変更で衰退した

### 目的
- 錬金術を活かして村興しに貢献する
- 1年間で村をどこまで発展させられるか

### 初期状態の村
- 村長の家
- 主人公の工房（自宅）
- 村人の家（数軒）
- よろず屋（非特化商店）1軒

---

## 3. ゲーム期間とゴール

- **期間**: 1年間（360日 = 12ヶ月 × 30日）
- **ゴール**: 最終日の評価でエンディング分岐（村発展度が主軸）

## 4. システム詳細

### 4.1 時間管理

| 行動 | 消費日数 | 備考 |
|------|----------|------|
| 調合 | 1〜5日 | レシピ難易度による |
| 依頼確認/受注/納品 | 0日 | |
| 採取隊派遣 | 0日 | 帰還まで別途待機 |
| ショップ | 0日 | |
| 休息 | 1日 | 体力全回復 |
| 勉強 | 3〜7日 | レシピ習得 |

### 4.2 調合システム

#### 流れ
1. レシピ選択
2. 素材選択（各素材に品質あり）
3. 調合実行 → 日数経過
4. 成功/失敗判定
5. 完成品の品質決定

#### 品質（1〜100）
- 素材・完成品すべてに品質がある
- 完成品品質 = f(素材平均品質, スキルLv, 乱数)
- 高品質 = 高値売却、依頼評価向上

#### 錬金術レベル
- 調合で経験値獲得 → レベルアップ
- 効果: 成功率UP、品質上限UP、レシピ解禁

### 4.3 素材収集（採取隊システム）

プレイヤーは直接採取しない。採取隊を雇って派遣する。

**派遣時**
- 派遣先選択（森/山/湖など）
- 派遣日数選択（3/5/7日）
- 費用支払い（日数×エリア単価）

**帰還時**
- エリアに応じた素材をランダム入手
- 長期派遣ほど量が多い
- 低確率でレア素材

### 4.4 依頼システム

#### 依頼タイプ
| タイプ | 内容 | 報酬傾向 |
|--------|------|----------|
| 納品 | 指定アイテム納品 | 標準 |
| 品質指定 | 品質○○以上を納品 | 高め |
| 大量納品 | 同アイテム複数納品 | 量に比例 |

#### ルール
- 同時受注: 最大3件
- 期限切れ: 名声ダウン
- 連続失敗: 依頼出現率ダウン

### 4.5 経済

**収入**
- 依頼報酬
- アイテム売却

**支出**
- 素材購入
- 採取隊派遣費
- レシピ本購入

## 6. 村発展システム

### 村発展度（0-100）
村の発展度合いを示す主要パラメータ。プレイヤーの活動によって上昇する。

### 上昇条件
| 行動 | 上昇量 | 備考 |
|------|--------|------|
| 依頼完了 | +1〜3 | 依頼の難易度による |
| 高品質納品（70+） | +1（追加） | 品質70以上で追加ボーナス |
| 連続成功ボーナス | +1（追加） | 5件連続成功ごと |
| 特別イベント | +5〜10 | ストーリーイベント |

### 発展段階と解放

| 発展度 | 段階 | 解放される要素 |
|--------|------|---------------|
| 0-9 | 寂れた村 | 基本機能のみ |
| 10-19 | 小さな村 | よろず屋の品揃え拡大 |
| 20-29 | 成長する村 | 採取隊（冒険者ガルドが移住） |
| 30-49 | 活気ある村 | 鍛冶屋（トーマスが開業）、依頼種類増加 |
| 50-69 | 繁栄する村 | 行商人（ミーナが定期訪問）、レア素材 |
| 70-89 | 名の知れた村 | 高難度依頼、遠方からの客 |
| 90-100 | 伝説の村 | 特別イベント |

## 8. バランス目安

### 初期状態
- 所持金: 500
- 錬金術Lv: 1
- 名声: 0
- 村発展度: 0
- 体力: 100
- 初期素材: ハルマム草×5, 清水×3

### 経験値テーブル（仮）
| Lv | 必要累計Exp |
|----|-------------|
| 1→2 | 100 |
| 2→3 | 250 |
| 3→4 | 500 |
| ... | ... |
| 19→20 | 50,000 |

### 調合経験値
- 基本: 難易度 × 10
- 成功ボーナス: +50%
- 高品質ボーナス: 品質70以上で +20%

### 村発展ペース目安
- 1年間（360日）で100到達は困難だが可能
- 普通にプレイして50〜70程度
- 真ED狙いは効率的なプレイが必要
