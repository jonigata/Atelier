# BGM設計

## 全体のコンセプト

**素朴な中世ヨーロッパの田舎 × ほのかな魔法の気配 × 成長の物語**

楽器の基本パレット：アコースティックギター、フルート、ピアノ、ハープ、弦楽四重奏、アコーディオン

---

## 楽曲リスト

### 1. タイトル画面 — 「小さな工房の朝」
- オルゴール風のピアノ + ハープのアルペジオ
- ゲームのメインテーマをシンプルに提示
- 穏やかだが、どこかワクワクする予感を含む

### 2. オープニング／プロローグ — 「旅立ちの馬車」
- アコースティックギターのアルペジオから始まり、フルートがメロディを奏でる
- 馬車に揺られて未知の村へ向かう期待と不安
- 後半でストリングスが加わり、壮大さが増す

### 3. 朝のフェーズ（日常BGM・メイン）— 「フォンテ村の日々」
- アコーディオン + フルート + 軽いパーカッション
- ヨーロッパの村の朝市を思わせる、温かく少し陽気な曲
- ループしても飽きない、心地よい日常感
- 季節でアレンジを変える案あり（春は明るめ、冬は落ち着いた音色）
- **ファイル**: `static/bgm/city.mp3`

### 4. 調合シーン — 「大釜のワルツ」
- ピアノ主体 + ハープのグリッサンド + 木琴の点描
- 3拍子で釜がコトコト煮える感じ
- 神秘的だけど怖くない、「台所の魔法」のような親しみやすさ

### 5. 依頼受注／掲示板 — 「広場の掲示板」
- アコースティックギター + ピチカート弦楽
- テンポ良く、「さて何を手伝おうかな」という前向きな気持ち

### 6. 採取地（共通ベース + ロケーション差分）

| 採取地 | イメージ | 特徴 |
|--------|---------|------|
| 近くの森 | 木漏れ日のワルツ | フルート + 鳥の声のSE。明るく平和 |
| 枯泉の丘 | 失われた泉の記憶 | オカリナ + リバーブ深め。寂しくも神秘的 |
| 深森 | 古き森の声 | チェロ + 低いドローン。荘厳で少し緊張感 |
| 火山麓 | 鍛冶の大地 | パーカッション強め + 低音弦。力強い |
| 古代遺跡 | 忘れられた知恵 | ハープ + ベル + エコー。魔法的で幻想的 |

### 7. ショップ（よろず屋カリン）— 「カリンのよろず屋」
- マンドリン + アコーディオン
- 少しコミカルで人懐こい

### 8. マルコ来訪時 — 「旅商人の口笛」
- 口笛（ホイッスル）+ ギター + タンバリン
- 風来坊っぽい軽快さと胡散臭さ
- メインテーマのフレーズを少し崩したアレンジ

### 9. 勉強シーン — 「古い教本」
- ピアノソロ or チェンバロ
- 静かで知的。ページをめくる音が聞こえそうな

### 10. 休息 — 「星空のランプ」
- ハープ + 弦楽のピアニッシモ
- 夜の工房で一息つく安らぎ

### 11. ストーリーイベント（汎用）— 「語り部の手紙」
- ピアノ + バイオリン
- 感情的だが押しつけがましくない

### 12. リーネ関連イベント — 「白い花の約束」
- フルート + ピアノ
- 最初は距離感があり、関係が深まるにつれアレンジが温かくなる

### 13. 査察カウントダウン（7日前〜1日前）— 「査察の足音」
- 弦楽四重奏 + オーボエ + チェンバロ + ティンパニ
- 試験前夜のような不安と覚悟が入り混じった雰囲気
- 8小節ごとの鐘の音で日が過ぎていく感覚
- ニ短調、3拍子、テンポ96程度
- ループ前に一瞬だけ長調に解決し、決意の光を見せる

生成プロンプト:
```
中世ファンタジーRPGのBGM。錬金術師の弟子が、組合の査察を数日後に控えて緊張している場面。
試験前夜のような不安と覚悟が入り混じった雰囲気。

弦楽四重奏を土台に、オーボエがメインメロディを奏でる。
チェンバロが時計の刻みのようなリズムを刻み、時折ティンパニが低く鳴る。
8小節ごとに鐘の音が1回鳴り、日が過ぎていく感覚を出す。

ニ短調、3拍子、テンポ96程度。
チェロのオスティナートが焦燥感を生み、オーボエの旋律は落ち着こうとしつつも揺れ動く。
ループ前に一瞬だけ長調に解決し、決意の光を見せてから冒頭に戻る。

ループ可能、90〜120秒。打楽器はティンパニのみ。
温かみのある生楽器の音色、シンセ不使用。
```

### 14. 査察シーン — 「師匠組合の紋章」
- 弦楽四重奏 + ティンパニ（控えめ）
- 厳粛で格式高い。試験を受ける緊張感
- 結果発表時にテンポが変わる

### 15. ライバル・ヴィクト登場 — 「もう一人の錬金術師」
- チェロ主体 + ピアノ
- 自信に満ちていて少し挑発的。悪役のテーマではない

### 16. 施設建設イベント — 「村が変わる日」
- メインテーマのアレンジ（管弦楽版）
- 高揚感と達成感

### 17. 助手獲得（ドラフト）— 「小さな仲間たち」
- グロッケンシュピール + ハープ + フルート
- 妖精的でキラキラした音色

## 補足アイディア

- **季節変化**: 同じ曲でも春（明るい木管）→夏（活気あるパーカッション）→秋（温かい弦楽）→冬（静かなピアノ）とアレンジを変える
- **村発展度連動**: 村が発展するにつれ日常BGMに楽器が増えていく（序盤はギター1本→終盤でアンサンブルに）
- **環境音の活用**: 工房の釜のコポコポ音、村の鶏の声、森の小川など、BGMと環境音のレイヤー

## 推奨される楽器・音色

- 弦楽器: ヴァイオリン、チェロ（温かみ）
- 木管: フルート、オーボエ（田舎情緒）
- ピアノ: 知的で親密感
- ハープ: 魔法的・妖精的
- アコースティックギター: 素朴さ
- アコーディオン: ヨーロッパ的、素朴
- 鐘・ベル: 時の流れ
