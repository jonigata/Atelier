# テストページ一覧

コンポーネントの動作・演出確認用ページ。`src/routes/test/` 以下に配置。

サーバが起動していれば `http://localhost:5473/test/<name>` でアクセス可能。

## ページ一覧

| パス | 概要 |
|------|------|
| `/test/draw` | ドローダイアログ演出テスト（施設ドロー・助手ドロー） |
| `/test/inspection-event` | 査察イベント演出テスト（月別・グレード別カットシーン再生） |
| `/test/inspection` | InspectionTracker コンポーネントテスト |
| `/test/stamp` | スタンプエフェクトテスト |
| `/test/quest-profit` | 依頼利益率分析ツール |

## /test/draw

施設ドロー・助手ドローの演出フロー確認。

**操作:**
- 「施設クリア」→「施設ドロー発動」で施設カードの演出確認
- 「助手クリア」→「助手ドロー発動」で全NEW助手カードの確認
- 「助手セット (Lv1/2/MAX)」→「助手ドロー発動」でNEW/レベルアップ/MAX混在の確認
- 「演出スキップ」チェックで skipPresentation 動作確認

**確認ポイント:**
- バッジのpop-inアニメーション
- カード裏面の staggered スライドイン
- 3Dフリップ（裏→表）
- フリップ完了前のクリック無効
- 選択後の他カードフェード + 結果テキスト + 自動閉じ
- 演出スキップ時の即時選択待ち

## /test/inspection-event

査察カットシーン（評価演出）の再生テスト。

**操作:**
- 月とグレードを選んで「カットシーンのみ再生」/「フルイベント再生」
- S〜Dの各グレード演出（紙吹雪・パーティクル等）を個別確認

## /test/inspection

InspectionTracker HUDコンポーネントの表示テスト。

## /test/stamp

スタンプ押下エフェクトの確認。

## /test/quest-profit

依頼の利益率・素材コスト分析ツール。ゲームバランス調整用。
