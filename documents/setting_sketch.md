# 設定スケッチ

## ログライン
師匠組合から「半年で辺境の村の特産品を開発せよ」と命じられた駆け出し錬金術師の少女は、回復薬しか作れない腕前で「先生」と呼ばれる村に赴任し、期待外れの正体がバレる前に、気難しい村娘や胡散臭い旅商人と協力して村の窮地を救う一品を完成させなければ、錬金術師の免許と、やっと見つけた自分の居場所を失う。

---

## 必要要素

### 誰が

**主人公: 駆け出し錬金術師の少女**
- 師匠のもとで修行していたが、「座学ばかりで実践が足りない」と判断され派遣される
- 作れるもの: 回復薬、毒消しのみ（最初のレシピは「復習」名目で学習）
- 性格: 前向き、素直、やや抜けている。プレイヤーキャラクターとして尖りすぎない
- 欠落: 自分の錬金術に確信がない。「何を作りたいか」がわからない

**親友候補: 気難しい村娘**
- 村長の娘。村への愛着が強い反面、「田舎のつまらなさ」にうんざりしている面もある
- 最初は主人公を「どうせまたすぐ帰る都会の人」と警戒する
- 実は村の薬草や自然素材に詳しい（祖母が民間療法の担い手だった）
- 主人公と対照的に「知識はあるが活かし方を知らない」立場

**旅商人**
- 定期的に村を訪れる商人。胡散臭いが商売の腕は確か
- 主人公に素材を売りつける/買い叩く一方、ときどき重要な情報やレアな素材をもたらす
- 商売相手として主人公の成功を望んでいる（取引が増えるから）

**冒険者パーティー**
- 村の周辺を活動拠点にしている小規模パーティー（2〜3人）
- 主人公に採取依頼素材の調達を頼まれる立場。逆に主人公から回復薬等を購入する相互関係
- 危険な採取地への案内役としても機能

**村人たち**
- 村長（親友の父）: 穏やかだが村の将来を憂いている。主人公に期待をかける
- 鍛冶屋のおやじ: 無口だが面倒見がいい。錬成に必要な器具の修理をしてくれる
- パン屋のおばちゃん: おしゃべりで情報源。依頼の多くはここから始まる

**ライバル: 隣村の錬金術師（先輩格）**
- 主人公と同じ師匠組合の派遣錬金術師だが、キャリアは上
- 隣村に赴任中。半年後の査察で主人公と同時に評価される
- 主人公を見下しているが、直接的な妨害はしない。むしろ「どうせ無理でしょ」と余裕を見せる
- 存在自体がプレッシャー。隣村は順調に発展しているという噂が入ってくる

### どこで

**辺境の村「フォンテ村」（仮称）**
- 王都から馬車で数日かかる辺境の農村
- かつては「フォンテの泉」という薬効のある湧き水で知られていたが、泉が枯れて以来衰退
- 現在の人口は30人程度。若者が出て行き、高齢化が進行中
- 村の特徴: 泉は枯れたが、周辺の土壌には泉の成分が染み込んでおり、独特の薬草が自生する。村人はそれを「雑草」としか思っていない
- **村の独自性**: この土壌でしか育たない薬草群が存在し、それが特産品開発の鍵になる

**主人公の工房**
- 村長が空き家を改装して用意してくれたもの。最初はボロい
- ゲーム進行に伴い拡張・改装される
- 工房の充実度が主人公の成長を可視化する

### 何をする / なぜ / 障害 / 結果

**何をする**: 村の独自素材を使った特産品（錬金アイテム）を開発する
**なぜ**: 師匠組合の命令（免許維持のため）→ 村人のため → 自分のため（居場所）
**障害**:
1. 素人同然の腕前（回復薬しか作れない）
2. 村人の過大な期待と、実力が伴わないことによる信頼喪失の危機
3. 素材の知識不足（何がどこにあるか、何に使えるかわからない）
4. 隣村のライバルとの比較プレッシャー
5. 半年という時間制限
**結果**: フォンテ村固有の薬草を活用した特産品を完成させ、査察に合格する

## 成立条件

### 条件1: 師匠組合の派遣制度
- **これが成立するには**: 組合が若手を辺境に送る理由が必要
- **設定**: 師匠組合は王国から「地方振興補助金」を受けている。その条件として、若手錬金術師を地方に派遣し、各地の錬金術インフラを整備する義務がある。半年ごとの査察は補助金の使途監査を兼ねている
- **現状**: 成立する

### 条件2: 主人公が「回復薬しか作れない」のに派遣される
- **これが成立するには**: 師匠が素人を送る理由、または組合側の事情が必要
- **設定**: 師匠は主人公の「基礎力の高さ」を見抜いている。座学は優秀だが実践経験がゼロ。「現場で揉まれれば化ける」という判断で敢えて送り出した。また、辺境の村は不人気な赴任先であり、志願者がいなかった
- **現状**: 成立する

### 条件3: 村人が「錬金術師の先生」を期待する
- **これが成立するには**: 村が錬金術師を必要とする理由が必要
- **設定**: フォンテの泉が枯れて以来、村は主産業を失った。村長が組合に「錬金術で新しい特産品を」と陳情していた。何年も待ったのでその期待は大きい
- **現状**: 成立する

### 条件4: 半年で特産品を開発できる
- **これが成立するには**: ゲームとしての尺とシナリオの整合
- **設定**: 主人公の基礎力は高く（師匠のお墨付き）、あとは実践経験と新素材の知識。フォンテ村の独自薬草という「未開の素材」があるため、既存レシピではない新しい発見が可能。段階的にレシピを拡大し（依頼を通じた学習）、最終的に特産品に到達するルート
- **現状**: 成立する（ゲーム進行として自然）

### 条件5: ライバルが「助けてくれない」のに「来られる」
- **これが成立するには**: ライバルが物理的にアクセスできるが助ける動機がない理由
- **設定**: ライバルは隣村に赴任中。査察で同時に評価されるため、主人公を助ける＝自分の優位を減らすことになる。直接妨害はしないが、協力する理由もない。たまに素材の仕入れで主人公の村を通りがかり、嫌味を言う程度
- **現状**: 成立する

### 条件6: 親友候補の村娘との関係構築
- **これが成立するには**: 最初の対立と、協力に至る契機が必要
- **設定**: 村娘は過去にも「村を助ける」と言って来た人が何人もいて、全員途中で帰った経験がある。だから主人公も信用しない。しかし主人公が祖母の民間療法レシピに興味を示したことで、少し態度が軟化する。やがて「この子は本気でこの村にいる気だ」と認め、素材採取のパートナーになる
- **現状**: 成立する

## 整合性チェック

| 条件A | 条件B | 整合性 | 備考 |
|-------|-------|--------|------|
| 師匠組合の派遣制度 | 主人公が素人で派遣される | OK | 不人気赴任先＋師匠の「基礎力を見抜いた」判断で整合 |
| 村人の期待 | 主人公が回復薬しか作れない | OK | 期待とのギャップがコメディの原動力 |
| ライバルが来られる | 主人公が助けを呼べない | OK | 査察で競合関係にあるため協力の動機がない |
| 半年の時間制限 | 特産品開発が可能 | OK | 段階的なレシピ拡大＋固有素材の存在で到達可能 |
| 村娘が素材に詳しい | 主人公が知識不足 | OK | 相互補完の関係。協力の必然性 |
| 泉が枯れた村 | 独自素材が存在する | OK | 泉の成分が土壌に残り、独特の薬草が自生するという設定 |

## ゲームシステムとの整合メモ

- **錬成の繰り返し**: 依頼→素材調達→錬成→納品のサイクルが自然に組み込まれている
- **失敗の扱い**: 錬成失敗は素材ロスとして処理。プロット上の分岐条件にはしない
- **定期的な関門**: 月次の小査察（or 季節イベント）を設けることで、ゲームの節目とシナリオの節目を同期させる
- **レシピ拡大**: 依頼の達成→新素材の発見→新レシピの学習、という自然な拡大経路
- **工房の発展**: ゲーム内の工房アップグレードがシナリオ上の成長と連動
